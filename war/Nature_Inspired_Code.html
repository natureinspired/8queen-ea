The menu is designed as a solution to the 8-queens puzzle so the links in the table are structured so that no two are on the same row, column or diagonal. It uses AJAX to invoke an evolutionary algorithm which evolves an optimal solution in O(1) out of a possible 4,426,165,368 solutions. It finds the solution by randomly creating a population of possible solutions, selects the best solutions so far, makes them breed together to generate offspring and repeats this process until it evolves the optimal solution. The evolutionary algorithm I wrote is highly tuned to find the solution in under a second. My code for this web site is available at: http://github.com/natureinspired/8queen-ea
